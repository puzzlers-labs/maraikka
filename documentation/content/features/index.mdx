# Features Overview

Maraikka is packed with powerful features designed to make file encryption simple, secure, and efficient. Explore all the capabilities that make Maraikka the perfect choice for protecting your sensitive data.

## Core Features

### 🔒 **[AES Encryption](/features/encryption)**
Military-grade encryption using Advanced Encryption Standard (AES) with 256-bit keys.

**Key Benefits:**
- Industry-standard security
- PBKDF2 key derivation
- Secure random salt generation
- No password storage

**Use Cases:**
- Protecting sensitive documents
- Securing financial records
- Encrypting personal photos
- Safeguarding business data

---

### 📁 **[File Explorer](/features/file-explorer)**
Intuitive file management with modern interface design and powerful navigation.

**Features:**
- List and grid view modes
- Real-time file statistics
- Drag-and-drop support
- Context menu integration
- File type recognition

**Benefits:**
- Easy file organization
- Quick visual identification
- Efficient bulk operations
- Seamless workflow integration

---

### 🔍 **[File Preview](/features/preview)**
Preview files without permanently decrypting them, supporting multiple file formats.

**Supported Formats:**
- **Images**: JPG, PNG, GIF, WebP, SVG
- **Videos**: MP4, WebM, MOV, AVI
- **Documents**: PDF files with page navigation
- **Text**: TXT, MD, JSON, CSV, and more

**Advanced Features:**
- Zoom and pan for images
- Video playback controls
- PDF page navigation
- Encrypted file preview with password

## Advanced Features

### 🔐 **[Hardware Authentication](/features/hardware-auth)** (Premium)
> [!NOTE]
> **Premium Feature**: Requires a valid license key.

Replace passwords with secure hardware authentication using FIDO2/WebAuthn standards.

**Supported Devices:**
- **YubiKey**: Physical security keys
- **Touch ID**: macOS biometric authentication
- **Windows Hello**: Windows biometric authentication
- **FIDO2 Devices**: Any compatible authenticator

**Benefits:**
- More secure than passwords
- Faster authentication
- No password to remember
- Phishing-resistant

## Technical Specifications

### Security
- **Encryption**: AES-256 in CBC mode
- **Key Derivation**: PBKDF2 with SHA-256
- **Random Generation**: Cryptographically secure
- **Memory Management**: Secure memory clearing

### Performance
- **File Size Limit**: No practical limit
- **Concurrent Operations**: Multi-threaded processing
- **Memory Usage**: Optimized for large files
- **Disk I/O**: Efficient streaming operations

### Compatibility
- **File Systems**: NTFS, APFS, ext4, and more
- **File Types**: All file types supported
- **Network Drives**: Full support
- **External Storage**: USB, external drives
