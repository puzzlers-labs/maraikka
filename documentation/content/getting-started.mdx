# Getting Started

Welcome to Maraikka! This guide will help you get up and running with secure file encryption in just a few minutes.

## First Launch

When you first open Maraikka, you'll see the main interface with:

- **Sidebar**: Contains actions, current path, and file statistics
- **Main Area**: File explorer showing your selected directory contents
- **Empty State**: Prompts you to select a directory to get started

## Basic Workflow

### 1. Select a Directory

Start by selecting a directory containing files you want to work with:

1. Click **"Select Directory"** in the sidebar
2. Choose a folder from your file system
3. Maraikka will display all files and subdirectories

> [!TIP]
> **Pro Tip**: You can also use the keyboard shortcut `Cmd+O` (macOS) or `Ctrl+O` (Windows/Linux) to quickly select a directory.

### 2. Understanding the Interface

Once you've selected a directory, you'll see:

#### File List
- **Files**: Regular files with their icons and sizes
- **Directories**: Folders you can navigate into
- **Encrypted Files**: Files with `.enc` extension and lock icons

#### File Statistics (Sidebar)
- Total files count
- Number of directories
- Number of encrypted files
- Current directory path

#### View Options
- **List View**: Detailed file information with sizes and dates
- **Grid View**: Visual thumbnails for images and file icons

### 3. Your First Encryption

Let's encrypt your first file:

#### Encrypt a Single File
1. **Hover over any file** in the list
2. **Click the "Encrypt" button** that appears
3. **Enter a strong password** in the modal
4. **Click "Confirm"**

The file will be encrypted and renamed with a `.enc` extension.

#### Encrypt an Entire Directory
1. **Click "Encrypt Directory"** in the sidebar
2. **Enter a password** for all files in the directory
3. **Click "Confirm"**

All files in the directory will be encrypted with the same password.

> [!WARNING]
> **Remember Your Password**: Lost passwords cannot be recovered. Consider using a password manager.

### 4. Decrypting Files

To decrypt files:

#### Decrypt a Single File
1. **Hover over an encrypted file** (has lock icon)
2. **Click the "Decrypt" button**
3. **Enter the password** used for encryption
4. **Click "Confirm"**

#### Decrypt an Entire Directory
1. **Click "Decrypt Directory"** in the sidebar
2. **Enter the password** used for directory encryption
3. **Click "Confirm"**

## Password Best Practices

Maraikka includes a built-in password strength meter to help you create secure passwords:

### Strong Password Guidelines
- **Minimum 12 characters** (6 minimum required)
- **Mix of uppercase and lowercase** letters
- **Include numbers and symbols**
- **Avoid common words** or personal information

## File Preview

Maraikka includes a powerful preview system:

### Supported File Types
- **Images**: JPG, PNG, GIF, WebP, SVG
- **Videos**: MP4, WebM, MOV
- **Documents**: PDF files
- **Text**: TXT, MD, JSON, and more

### Using Preview
1. **Click any supported file** in the list
2. **Preview opens** in the right panel
3. **Use controls** to zoom, navigate, or close

### Encrypted File Preview
1. **Click an encrypted file**
2. **Enter the password** when prompted
3. **File decrypts temporarily** for preview only

> [!NOTE]
> Preview decryption is temporary - the original file remains encrypted.
