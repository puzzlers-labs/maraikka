## Before You Start

### Prerequisites
- Maraikka installed on your system ([Installation Guide](/installation))
- Basic understanding of file systems and directories
- A test directory with some files to practice with

### Recommended Reading Order
1. **Start with [Interface Guide](/user-guide/interface)** to understand the layout
2. **Practice [File Encryption](/user-guide/encryption)** with test files
3. **Learn [File Decryption](/user-guide/decryption)** to restore files
4. **Explore advanced features** like preview and editors
5. **Customize with [Preferences](/user-guide/preferences)**
